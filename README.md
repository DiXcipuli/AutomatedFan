# Smart Fan

Controlling 3 fan (propeller speed, and post rotation) through a RF remote

## Components
- Arduino Nano
- QIACHIP Wireless RX480E-4WQB 433Mhz RF Module Transmitter and Receiver
- L298N Dual H-Bridge Motor Drive Controller
- 3 DC motors
- 3 stepper motors
